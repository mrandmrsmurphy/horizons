### Matthean
1. Matthew 1:1 - GSM - anarthrous - This might be simply 'Jesus' + 'Christ'.
1. Matthew 1:18 - GSM - articular - If this is an instance of 'Jesus Christ' as a phrase, it would be the only one with the definite article.

### Markan
1. Mark 1:1 - GSM - anarthrous
Given the uniqueness of this verse, and the fact that Mark uses {{GW|XRISTOS}} seven times on a whole, it is unlikely that he is using this phrases as others do.  It is listed here, but also under {{GW|XRISTOS}}, unlike all other instances.
